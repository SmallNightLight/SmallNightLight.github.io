Super Monster Wheels



This game was created in 2 months in a team of 6 people. The team consisted out of 2 artists, 3 designers and 1 engineer (me). It was made in the unity game engine.


The theme of the game was "Monsters". We received 4 drawings with monsters from a local school that we needed to implement in a playable game.

We made a chaotic driving game, featuring these monsters and using wii motes (with steering wheel) as input device


I was responsible for:

  • Creating the toon rendering style (More info)
  • Creating a tooling pipeline for the designers (spline roads)
  • AI enemies
  • Handling local multiplayer
  • Custom input (Wii motes)
  • All gameplay code
  • ...

The project received the highest possible grade (10)
